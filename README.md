# create-nudocs

Scaffold a new NuDocs project.

## Usage

```bash
npm create nudocs@latest
```

You can also provide the target directory explicitly:

```bash
npm create nudocs@latest docs
```

## Options

- `-y`, `--yes`: skip prompts and use defaults
- `--github owner/repo`: set the GitHub repository in `.config/docs.yaml`
- `--install`: install dependencies after scaffolding
- `--no-install`: skip dependency installation
